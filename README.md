# AI-POWERED-SERVICE

An intelligent automation platform — beautiful demo website.

## 🧠 What's inside

| File | Description |
|------|-------------|
| `index.html` | Full landing page with hero, features, how-it-works, pricing, testimonials, and live demo |
| `css/styles.css` | Dark-theme CSS with glassmorphism, gradients, animations, and responsive breakpoints |
| `js/main.js` | Interactive chat simulation, scroll-reveal animations, counter ticks, demo form, mobile nav |

## 🚀 How to run

**Option 1 — Open directly**  
Just double-click `index.html` in your file browser.

**Option 2 — Simple HTTP server (recommended)**  

```bash
# Using Python 3
python3 -m http.server 8080

# Using Node (npx)
npx serve .
```

Then open `http://localhost:8080` in your browser.

## ✨ Features

- **Dark theme** with animated gradient blobs in the background  
- **Glassmorphism cards** for UI elements  
- **Animated chat** simulation in the hero section  
- **Live demo form** — type a request and get an AI-like response  
- **Scroll-reveal animations** via IntersectionObserver  
- **Animated counters** in stats section  
- **Fully responsive** — desktop, tablet, and mobile layouts  
- **Zero framework dependencies** — pure HTML/CSS/JS
